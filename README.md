## 🚀 Featured Projects

### 1. 🤖 [PrepMate — GenAI-Powered Career Accelerator](https://github.com/Mahadevj21/Prepmate) · [🌐 Live Demo](https://prepmate-snowy.vercel.app/)

A production-deployed, full-stack GenAI platform that bridges the gap between learning and employment using **Google Gemini LLMs**.

- **AI Roadmap Generation** — Enter a career goal (e.g. "Full Stack Dev in 3 months") and get a structured, week-by-week learning path generated by Gemini Flash
- **AI Mock Interviews** — Simulate real technical interviews for any stack (Java, Python, React, etc.); AI evaluates answers and gives a score with improvement tips
- **Sticky Key Rotation** — Proprietary algorithm that tracks up to 3 Gemini API keys, auto-fails over on `429` errors, and stays on a working key to maximize uptime on free-tier quotas
- **JWT Auth + CORS** — Stateless, secure authentication with strict origin mapping for production
- **Deployed on Vercel + Render + Supabase**

**Stack:** `React 19` · `Vite` · `Spring Boot 3` · `Java 17` · `PostgreSQL (Supabase)` · `Google Gemini API` · `JWT` · `Render` · `Vercel`

### 2. 🏡 [PropManage — Real Estate Management System](https://github.com/Mahadevj21/RealEstate)

A full-stack real estate platform with role-based dashboards for **Admins**, **Sellers**, and **Buyers**.

- **Buyers** can browse listings, filter by price/location/type, view properties on an interactive map (Leaflet), and purchase via a digital wallet
- **Sellers** can list properties with dynamic image uploads, manage incoming offers, and track wallet earnings
- **Admin** controls user access, moderates listings, collects brokerage fees, and views platform analytics (Recharts)

**Stack:** `React.js` · `Spring Boot 3` · `Java 17` · `PostgreSQL` · `Hibernate/JPA` · `React-Leaflet` · `Recharts` · `Maven`

### 3. 📋 [JobTracker — Career Command Center](https://github.com/Mahadevj21/job-application-tracker-api)

A full-stack SaaS app to turn the job hunt into a structured, data-driven workflow.

- Tracks applications with company, role, salary, and status — all in one place
- Manages multiple interview rounds with date tracking and notes per round
- Auto-sends follow-up emails when an application goes cold (Spring Boot Mail + SMTP)
- Real-time analytics dashboard showing pipeline health and interview success rates
- JWT-secured auth with a modern dark-mode UI using glassmorphism + Tailwind CSS

**Stack:** `React 18` · `Vite` · `Tailwind CSS` · `Spring Boot 4` · `Java 17` · `PostgreSQL` · `Spring Security (JWT)` · `Spring Mail`
